# <img src="assets/icon.webp" width="40" vertical-align="middle"> バンド解析 (CarrierBand Analyzer)

---

日本の携帯キャリア（ドコモ・au・SB・楽天）及びサブブランドに特化した、周波数帯（バンド）可視化・解析Androidアプリです。

## 📱 特徴

* **リアルタイム可視化:** 接続中のLTE/5Gバンド（例：Band 1, Band 19, n78）を表示します。
* **キャリア最適化:** 日本の主要4キャリアおよびサブブランドを自動判定します。
* **プラチナバンド判定:** 繋がりやすさを左右する「プラチナバンド」(B8, B18, B19, B28) を強調表示します。
* **バックグラウンド監視:** アプリを閉じていても移動中のバンドを記録し続けます。
* **多言語対応 (7カ国語):** 日本語、英語、スペイン語、ドイツ語、ロシア語、中国語、韓国語に対応。アプリ内の設定からいつでも言語を切り替え可能です。

### 📤 共有・エクスポート
- **レポート共有**: 解析結果をテキスト形式でクリップボードにコピーしたり、SNSへシェアできます。
- **CSVエクスポート**: 観測された詳細なバンド履歴（日時、キャリア、バンド）をCSVファイルとして出力し、外部アプリ（メール、ドライブ等）で保存・分析できます。

## 📡 対応バンド (日本国内)

| カテゴリ | バンド | 説明 |
| :--- | :--- | :--- |
| **プラチナ** | B8, B18, B19, B26, B28 | エリアカバー重視 (山間部・屋内) |
| **主力 (Main)** | B1, B3, B11, B21 | 速度・容量重視 |
| **高速 (4G)** | B41, B42 | 超高速LTE |
| **転用5G** | n1, n3, n28 | 4G周波数を5Gに転用 (エリア拡大用) |
| **5G (Sub6)** | n77, n78, n79 | 標準的な5G |
| **5G (ミリ波)** | n257 | 超高速5G |

## 🌍 海外利用・未対応キャリアについて

本アプリは世界共通の規格（PLMN/Android API）に準拠しています。

* **✅ 利用できる機能:** 海外キャリアや未定義の回線であっても、**「現在接続中のバンド」や「キャリア名」のリアルタイム表示（モニター機能）**は可能です。海外旅行時の通信確認ツールとしてご利用いただけます。
* **⚠️ 利用できない機能:** 「バンド網羅率（スコア）」や「有利・不利」の自動判定機能は、現在**日本の主要キャリア（MNO/MVNO）に最適化**されています。海外利用時はこれらの分析スコアが表示されない場合があります。
