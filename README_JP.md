日本の携帯キャリア（ドコモ・au・SB・楽天）及びサブブランドに特化した、周波数帯（バンド）可視化・解析Androidアプリです。

## 📱 特徴

* **リアルタイム可視化:** 接続中のLTE/5Gバンド（例：Band 1, Band 19, n78）を表示します。
* **キャリア最適化:** 日本の主要4キャリアおよびサブブランドを自動判定します。
* **プラチナバンド判定:** 繋がりやすさを左右する「プラチナバンド」(B8, B18, B19, B28) を強調表示します。
* **バックグラウンド監視:** アプリを閉じていても移動中のバンドを記録し続けます。

## 🛠 使用技術

* **言語:** Kotlin
* **UI:** Jetpack Compose (Material Design 3)
* **アーキテクチャ:** MVVM

## 📡 対応バンド

| カテゴリ | バンド | 説明 |
| :--- | :--- | :--- |
| **プラチナ** | B8, B18, B19, B26, B28 | エリアカバー重視 (山間部・屋内) |
| **主力 (Main)** | B1, B3, B11, B21 | 速度・容量重視 |
| **高速 (4G)** | B41, B42 | 超高速LTE |
| **5G (Sub6)** | n77, n78, n79 | 標準的な5G |
| **5G (ミリ波)** | n257 | 超高速5G |
